## 01 — Identidad Core & Diferenciadores
Selva Digital es desarrollo web freelance de código propio y pago único para PyMEs argentinas.

### Diferenciadores Clave:
- **Pago único**: Primer año de dominio + hosting incluidos. Sin suscripciones infinitas.
- **Trato Directo**: Trato directo con Christian Speziali (diseña, programa y da soporte).
- **Código propio**: Nada de templates genéricos saturados.
- **Capacidad limitada**: Máximo 3 proyectos simultáneos (exclusividad real).

## 02 — Identidad Visual (Especificaciones Técnicas)

### Paleta de Colores Cerrada
- **Acento**: `#2BB673` (Verde Selva) -> Usar **únicamente** para botones de acción (CTA), links destacados, highlights y badges. Nunca para texto de cuerpo.
- **Hover Acento**: `#5EE6B7` (Verde claro para transiciones y hovers).
- **Fondo Oscuro**: `#0A0B0D` (Negro Profundo).
- **Fondo Claro**: `#FAFAFA` (Blanco Crudo).
- **Contenedores (Surface)**: `#121316` / `#1A1C20` (Negro Slate para cards sobre fondo dark).
- **Errores**: `#FF6B6B` (Rojo).
- **Warnings**: `#FFB547` (Ámbar).

### Tipografía
- **Títulos / Display**: **Geist** (pesos 600/700) -> Aporta aspecto limpio y técnico.
- **Cuerpo / UI**: **Inter** (pesos 400/500) -> Máxima legibilidad.
- **Detalles / Datos / Monospace**: **JetBrains Mono** -> Usar para precios, tags, timestamps y etiquetas estilo terminal.

### Componentes & Espaciado
- **Radios de Esquina (Border-radius)**:
  - Chips / Badges: `6px`
  - Botones / Inputs: `10px`
  - Cards / Contenedores: `12px`
  - Hero sections: `20px`
- **Botones**:
  - **Primario**: Fondo verde `#2BB673` con texto `#06140C`.
  - **Secundario / Ghost**: Sin fondo, con borde del color de acento y texto del color de acento.
  - *Restricción:* Nunca usar botones grises o con colores que no pertenezcan a la paleta.
- **Espaciados**: Múltiplos de `8px` (`8px`, `16px`, `24px`, `32px`, `48px`).

## 03 — Identidad Verbal & Tono de Voz

### Personalidad de Marca
- **Directa e informal**: Sin palabras corporativas vacías como "soluciones integrales".
- **Coloquial Argentina**: Utilizar español rioplatense natural (ej: *"laburar"*, *"te marea"*, *"te la mandás directo"*, *"arrancar"*).
- **Técnica pero entendible**: Explicar la tecnología en términos de beneficios de negocio.
- **Enfoque en Transformación**: El cliente no compra características de código; compra recuperar tiempo y dejar de atender consultas manuales por WhatsApp todo el día.

### Reglas para Redacción
1. **Frases cortas** y uso de **negrita** para enfatizar conceptos clave.
2. **Prueba social integrada**: Acompañar los textos de números de casos reales (ej: *"+34% leads"*, *"CTR x2.1"*).
3. **Escasez Real**: Usar etiquetas de disponibilidad tipo terminal: `PRÓXIMA VENTANA: JUNIO 2026`, `2/3 CUPOS TOMADOS`.
4. **CTAs Directos**: Siempre terminar con flecha: `Pedir presupuesto →`, `Ver portfolio →`.
5. **Ahorro de Tiempo y Dinero**: Cada post o copy debe justificar claramente cuánto tiempo le devuelve al cliente o cuánto dinero le ayuda a ganar/ahorrar.
